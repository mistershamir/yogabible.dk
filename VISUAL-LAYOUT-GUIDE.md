## 📱 Responsive Breakpoints

### Desktop (980px+)
- Full header navigation with dropdowns
- 4-column trust badges grid
- 2-column program cards
- 3-column courses grid
- 3-column instructors grid
- 3-column testimonials grid

### Tablet (768px-979px)
- Hamburger menu
- 2-column trust badges grid
- 2-column program cards
- 2-column courses grid
- 2-column instructors grid
- 2-column testimonials grid

### Mobile (<768px)
- Hamburger menu with drawer
- 1-column trust badges grid
- 1-column program cards (stacked)
- 1-column courses grid (stacked)
- 1-column instructors grid
- 1-column testimonials grid
