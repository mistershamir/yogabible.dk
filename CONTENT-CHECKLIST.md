## 📊 Image Optimization Tips

Before uploading images:

1. **Resize to exact dimensions**
   - Don't use larger images (slow page load)
   - Use Photoshop, Preview, or online tools

2. **Compress files**
   - Target: <200KB per image
   - Use TinyPNG.com or Squoosh.app
   - Quality: 80-85%

3. **Use correct format**
   - Photos: JPG
   - Icons/logos: SVG or PNG
   - If WebP supported: Use WebP with JPG fallback

4. **Name files clearly**
   - Good: `inversions-handstand-course.jpg`
   - Bad: `IMG_1234.jpg`
